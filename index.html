Create an angular application which has two pages.

  • Home Page
  • Data Page

Home Page will have link to Data Page. By default Home Page will open. Directly pasting URL for Data Page will open Data Page. Data page will display and unordered(nested) list or products. Product List items will be fetched from server using HTTP API.

Here is Demo site to be created

Data API Links:

https://harshveersingh.github.io/ia/phones.json
https://harshveersingh.github.io/ia/laptops.json

Product Item Schema is: 
Item = {
	Name: string,
	Children: Item[]
}
